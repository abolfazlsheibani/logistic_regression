# Logistic Regression Customer Prediction

A simple Machine Learning project using Scikit-Learn to predict whether a customer will purchase a product based on their age and estimated salary.

## 🚀 Features
- Uses **Pandas** for data management.
- Implements **Logistic Regression** for binary classification.
- Predicts both the **Class** (0 or 1) and the **Probability** of purchase.

## 🛠️ Requirements
To run this code, you need to install the following libraries:
```bash
pip install pandas scikit-learn numpy
```

## 💻 Usage
1. Clone the repository.
2. Run the main script:
```bash
python logistic_regression.py
```

## 📊 Dataset
The model is trained on a sample dataset including:
- **Age**: The person's age.
- **EstimatedSalary**: The person's annual income.
- **Bought**: Target variable (1 = Purchased, 0 = Not Purchased).
